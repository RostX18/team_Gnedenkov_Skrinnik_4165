# Прогноз действия игрока по истории ходов

Цель проекта - научить модель предсказывать следующий ход соперника и, возможно, самой ходить наилучшим образом.

## О проекте

Данный проект направлен на создание модели, предсказывающей следующий ход соперника в шахматах, и выбирающей наилучший ход из возможных.
Итоговый проект составляет собой модель, способную предугадывать следующий ход и знать свой.
Пока проект находится в начальной стадии, а именно - простые модели, работа с небольшим набором данных.


## Используемые технологии

- Python 3.10+

## Архитектура проекта (простая схема)

LSTM - модель с долгой краткосрочной памятью - тип рекуррентной нейронной сети (RNN), способный обучаться долговременным зависимостям.
